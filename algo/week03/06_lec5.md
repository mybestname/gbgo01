### 二叉树
- Binary Tree
  - 任意节点，子节点不超过2。 （0，1，2）
- 满二叉树
  - 所以层里面，每个节点都是2个子节点。

### 二叉树的遍历
- 前序：根-左-右  A BDHIEJ CFG 注意左代表左子树，右代表右子树。
- 中序：左-根-右  HDIBJE A FCG 
- 后序：左-右-根  HIDEJB FGC A
- 层次序：A BC DEFG HIJ

```
              A
            /  \
           B    C
         / \   / \  
       D    E F  G
     / \   /  
   H   I  J 
```
- 前、中、后都是深度优先的顺序。
- 前、中、后序的遍历一般采用**递归**求解
- 层次序一般采用**队列**求解
- 树的层次遍历又称树的**广度优先遍历**。
