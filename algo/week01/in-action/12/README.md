## 304. 二维区域和检索 - 矩阵不可变
- https://leetcode-cn.com/problems/range-sum-query-2d-immutable/
- 给定一个二维矩阵，计算其子矩形范围内元素的总和，该子矩阵的左上角为 (row1, col1) ，右下角为 (row2, col2) 。
```
 3  0   1   4  2
 5  6   3   2  1
 1 '2' '0' '1' 5
 4 '1' '0' '1' 7
 1 '0' '3' '0' 5
```
- 上图子矩阵左上角 (row1, col1) = (2, 1) ，
  右下角(row2, col2) = (4, 3)，
  该子矩形内元素的总和为 8。
-  `2+0+1+1+0+1+0+3+0 = 8`  


```
示例：

给定 matrix = [
[3, 0, 1, 4, 2],
[5, 6, 3, 2, 1],
[1, 2, 0, 1, 5],
[4, 1, 0, 1, 7],
[1, 0, 3, 0, 5]
]

sumRegion(2, 1, 4, 3) -> 8
sumRegion(1, 1, 2, 2) -> 11
sumRegion(1, 2, 2, 4) -> 12
```

### 思路：
-构造前缀和矩阵。(使用公式1)
-使用公式2来计算子矩阵和。
